# **App Name**: Weather Parader

## Core Features:

- Location Input: Enables users to input locations, including current location, custom locations, and journey waypoints.
- Weather Prediction: Generates weather predictions based on the user-inputted location.
- Probabilistic Analysis Tool: Employs a tool powered by generative AI to express weather forecasts as probabilities.
- Hourly and Daily Forecasts: Displays hourly and daily weather forecasts, including temperature, precipitation, and wind.
- Interactive Map: Renders a map showing locations and associated weather conditions, including weather pattern visualizations and journey route weather pins.
- Weather Risk Score Tool: Uses generative AI to calculate and display a weather risk score badge for each journey timepoint, accompanied by an explanation.

## Style Guidelines:

- Primary color: Vivid blue (#4285F4), symbolizing clarity and trustworthiness.
- Background color: Light gray (#F5F5F5), providing a neutral and clean backdrop.
- Accent color: Warm orange (#FF8A65), used to highlight important data and calls to action.
- Headline font: 'Playfair', serif, for a touch of elegance. Using 'PT Sans' as body font.
- Body font: 'PT Sans', sans-serif, for readability and modern look. Using 'Playfair' for the page heading.
- Use outlined icons from 'Lucide-react or Heroicons', simple and easily understandable.
- Subtle transitions for map interactions and loading states.
